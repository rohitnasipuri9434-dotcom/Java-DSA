## 🚀 How I Practice

1. **Understand:** Read the constraints (they usually hint at the required complexity).
2. **Brute Force:** Get a working solution first, no matter how slow.
3. **Optimize:** Refine the logic using appropriate data structures.
4. **Reflect:** Look at the top-performing solutions to see what I missed.
